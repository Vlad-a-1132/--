## Что нужно настроить 🔧

### 1. Создайте файл `.env.local` в корне проекта:

```bash
# Database (у вас уже есть)
DATABASE_URL="postgresql://username:password@localhost:5432/equilibrium_db"

# NextAuth (у вас уже есть)
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# Email configuration (ДОБАВЬТЕ ЭТО)
EMAIL_USER="your-email@gmail.com"
EMAIL_PASS="your-app-password"
```

### 2. Настройте Gmail для отправки email:

1. Включите двухфакторную аутентификацию в Google
2. Создайте App Password в настройках безопасности
3. Используйте этот пароль в `EMAIL_PASS`

## 🧪 Тестирование

### Тест 1: Регистрация
1. Перейдите на `http://localhost:3000/register`
2. Заполните форму и зарегистрируйтесь
3. Проверьте email с кодом подтверждения

### Тест 2: Подтверждение email
1. Скопируйте код из email
2. Перейдите на `http://localhost:3000/login`
3. Введите email и пароль
4. Введите код подтверждения

### Тест 3: Профиль
1. После входа вы попадете на `/profile`
2. Попробуйте отредактировать имя
3. Попробуйте сменить пароль

## 🐛 Возможные проблемы

### Проблема: "Ошибка отправки email"
**Решение:** Проверьте настройки Gmail и переменные окружения

### Проблема: "База данных не синхронизирована"
**Решение:** Запустите `npx prisma migrate dev`

### Проблема: "NextAuth ошибка"
**Решение:** Проверьте `NEXTAUTH_SECRET` и `NEXTAUTH_URL`
