# Система аутентификации и профиля пользователя

## Обзор

Реализована полноценная система регистрации, авторизации и управления профилем пользователя с подтверждением email.

## Основные возможности

### 🔐 Регистрация
- Регистрация новых пользователей с валидацией данных
- Автоматическая отправка кода подтверждения на email
- Код подтверждения действителен 10 минут
- Возможность повторной отправки кода

### ✅ Подтверждение email
- 6-значный код подтверждения
- Автоматическая активация аккаунта после подтверждения
- Отправка приветственного email после подтверждения

### 🔑 Авторизация
- Вход по email и паролю
- Проверка активности аккаунта
- Автоматическое перенаправление на профиль

### 👤 Профиль пользователя
- Просмотр и редактирование личной информации
- Смена пароля с подтверждением
- Красивый и современный интерфейс
- Выход из аккаунта

## Настройка

### 1. Переменные окружения

Создайте файл `.env.local` в корне проекта:

```bash
# Database
DATABASE_URL="postgresql://username:password@localhost:5432/your_database"

# NextAuth
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# Email configuration (Gmail)
EMAIL_USER="your-email@gmail.com"
EMAIL_PASS="your-app-password"
```

### 2. Настройка Gmail

1. Включите двухфакторную аутентификацию в Google аккаунте
2. Создайте App Password:
   - Перейдите в настройки безопасности Google
   - Найдите "Пароли приложений"
   - Создайте новый пароль для приложения
   - Используйте этот пароль в `EMAIL_PASS`

### 3. Установка зависимостей

```bash
npm install nodemailer @types/nodemailer
```

### 4. Миграция базы данных

```bash
npx prisma migrate dev --name add-email-verification
```

## Использование

### Регистрация нового пользователя

1. Перейдите на `/register`
2. Заполните форму: имя, email, пароль
3. Нажмите "Зарегистрироваться"
4. Проверьте email и введите код подтверждения
5. После подтверждения аккаунт активируется

### Вход в систему

1. Перейдите на `/login`
2. Введите email и пароль
3. Если аккаунт не подтвержден, появится форма ввода кода
4. После успешного входа перенаправление на профиль

### Управление профилем

1. Перейдите на `/profile`
2. Нажмите "Редактировать" для изменения данных
3. Нажмите "Сменить пароль" для смены пароля
4. Все изменения сохраняются автоматически

## Безопасность

- Пароли хешируются с помощью bcrypt
- Коды подтверждения имеют ограниченное время действия
- Все API защищены аутентификацией
- Валидация данных на клиенте и сервере
- Защита от SQL-инъекций через Prisma
